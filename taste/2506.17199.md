# Tighter Error Bounds for the qDRIFT Algorithm

**arXiv:** [2506.17199](https://arxiv.org/abs/2506.17199)

**Authors:** I.J. David, I. Sinayskiy, F. Petruccione

**Submission Date:** June 20, 2025

---

## Abstract

Improved qDRIFT error bounds using Jensen's inequality, achieving linear (rather than quadratic) scaling in the coefficient sum for certain regimes.

---

## Key Results

### Improved Error Bound
**Original qDRIFT (Campbell 2019):**
$$\epsilon \leq \frac{2\lambda^2 t^2}{N}$$

**This work:** Tighter constants and improved scaling using Jensen's inequality

### Applicability
- Closed quantum systems (Hamiltonian evolution)
- Open quantum systems (Lindbladian dynamics)
- Removes dependence on maximum diamond norm for open systems

### Applications Demonstrated
1. Quantum chemistry simulations
2. Dissipative transverse field Ising model
3. Hamiltonian encoding for quantum ML
