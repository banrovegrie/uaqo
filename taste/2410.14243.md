# Explicit Error Bounds with Commutator Scaling for Time-Dependent Product and Multi-Product Formulas

**arXiv:** [2410.14243](https://arxiv.org/abs/2410.14243)

**Authors:** Kaoru Mizuta, Tatsuhiko N. Ikeda, Keisuke Fujii

**Submission Date:** October 18, 2024

---

## Abstract

First explicit commutator-scaling error bounds for generic time-dependent Hamiltonians using novel Floquet embedding technique.

---

## Key Results

### Main Error Bound
$$\epsilon \sim \mathcal{O}\left(\sum_{\text{commutators}} \|[H_j(t), [H_k(t), \ldots]]\| + \|\partial_t H_j(t)\| \cdot (\Delta t)^{p+1}\right)$$

### Novel Technique: Floquet Embedding
- Embeds time-dependent Hamiltonians into time-periodic systems
- Maps time-dependent error analysis to time-independent formulations
- Unified treatment for both cases

### System Size Scaling

| Interaction Type | Scaling |
|------------------|---------|
| Finite-range | O(n) |
| Short-range (power-law) | O(n · polylog(n)) |
| Long-range | O(n^α), α > 1 |

### Extensions
- Multi-product formulas for time-dependent systems
- Applications to Floquet engineering, adiabatic preparation
